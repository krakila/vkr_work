# ИС анализа стресса игроков

## Запуск сервера
```bash
python server_app.py
```

## Запуск клиентского приложения
```bash
python desktop_client.py
```

## Тесты
```bash
pytest -q
```

## Что входит
- регистрация и авторизация;
- роли: игрок и исследователь;
- клиент-серверная архитектура на REST API;
- ожидание активного окна игры перед стартом сессии;
- хранение данных в SQLite;
- экранная запись и экспорт медиа;
- просмотр сессий с синхронизированными графиками;
- сравнение игроков по трём группам заболеваний: здоровые, астма/лёгкие, сердечные;
- rule-based калибровка и отдельная ML-модель.
